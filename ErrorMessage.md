# Error Message

## User Manager

* OK

```json
{
  "code": 100,
  "message": "OK"
}
```

* Username or Password Error

```json
{
  "code": 101,
  "message": "username_or_password_err"
}
```

* Username or Password Invaild

```json
{
  "code": 102,
  "message": "username_or_password_invaild"
}
```

* Username Duplicated

```json
{
  "code": 103,
  "message": "username_duplicated"
}
```

* Session expired

```json
{
  "code": 104,
  "message": "session_expired"
}
```
